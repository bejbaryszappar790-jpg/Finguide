# Finguide AI MVP

Рабочий фронтенд MVP для AI-финансового помощника.

## Что внутри

- Backend на чистом Node.js без внешних зависимостей
- REST API с роутами `/api/health`, `/api/auth/*`, `/api/chat`, `/api/business/tax`, `/api/mentor/plan`
- Регистрация и логин через backend MVP-сессии
- 3 AI-раздела:
  - универсальный чат
  - бизнес партнер для налогов, НДС, маржи и решений
  - финансовый ментор для физического лица
- Настоящая AI-интеграция через Google Gemini API
- Разные personas для трех AI-ассистентов
- Русский и казахский языки
- Калькулятор НДС, налога на прибыль, чистой прибыли и маржи
- Личный финансовый план с рекомендациями
- Адаптивный интерфейс для ноутбука и телефона

## Как запустить

```bash
cd /Users/beybaris/Documents/Finguide/finguide-mvp
GEMINI_API_KEY=ваш_ключ npm start
```

Потом откройте:

```text
http://localhost:4173
```

Если `GEMINI_API_KEY` не задан, приложение запустится, но `/api/chat` вернет ошибку `ai_not_configured`.

## Тесты

```bash
npm test
```

Тесты проверяют API-роуты, бизнес-расчеты, персональный план, валидацию и AI-интеграцию с mock Gemini API.
