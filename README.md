Finguide AI MVP

Finguide AI — это инновационная финтех-платформа и MVP умного финансового помощника. Проект объединяет стильный современный интерфейс с мощными AI-возможностями для помощи как физическим лицам, так и бизнесу в решении их финансовых задач.

✨ Ключевые возможности

3 специализированных AI-ассистента:
🤖 Универсальный ассистент — ответы на общие вопросы по финансам и планированию.
🏢 Бизнес-партнер — помощь предпринимателям с налогами, юнит-экономикой, расчетом маржи и принятием решений.
💰 Финансовый ментор — помощь физическим лицам с планированием бюджета, закрытием долгов и накоплениями.
Интеграция с Groq API — использование быстрых и мощных open-source LLM (например, LLaMA 3) для генерации ответов.
Двуязычный интерфейс — поддержка русского (ru) и английского (en) языков.
Финансовые калькуляторы — встроенные расчеты НДС, налога на прибыль, чистой прибыли и маржинальности.
Премиальный Fintech-дизайн — современный UI с эффектами Glassmorphism, мягким свечением (glow-эффекты) и строгим контролем скролла (100vh) для создания ощущения полноценного нативного приложения.
🛠 Технологический стек

Frontend: Vanilla HTML5, CSS3 (Custom Properties, Grid/Flexbox, Glassmorphism), Vanilla JavaScript. Никаких тяжелых фреймворков.
Backend: Чистый Node.js (node:http) без Express, встроенный роутинг.
AI Provider: Groq API (OpenAI-совместимый формат запросов).
Тестирование: Встроенный Node.js Test Runner (node:test).
🚀 Как запустить локально

1. Установка зависимостей

Убедитесь, что у вас установлен Node.js (версия 18+).

cd /Users/beybaris/Documents/Finguide/finguide-mvp
npm install
2. Настройка переменных окружения

Для работы AI необходимо получить бесплатный API-ключ от Groq Cloud.

Создайте файл .env или экспортируйте переменную напрямую в терминале:

export GROQ_API_KEY="ваш_ключ_от_groq_api"
💡 Если запустить проект без GROQ_API_KEY, приложение продолжит работу, но ИИ будет возвращать мок-ответы (заглушки) для тестирования интерфейса.
3. Запуск сервера

npm start
Приложение будет доступно по адресу: http://localhost:4173

🧪 Тестирование

В проекте настроено модульное тестирование API-роутов, бизнес-расчетов и AI-интеграции (с моками для Groq).

Для запуска тестов выполните:

npm test
📂 Структура проекта

app.js — основная логика фронтенда (состояния, роутинг, запросы к API, управление DOM).
styles.css — стили проекта, включая систему переменных, анимации и fintech decor.
index.html — основная разметка приложения (App Shell).
server/ — исходный код бэкенда.
index.js — точка входа сервера.
routes/api.js — обработка всех /api/* эндпоинтов.
services/ — бизнес-логика (расчет налогов, промпты агентов, интеграция с OpenAI/Groq).
tests/ — unit-тесты для бэкенда.
Сделано с фокусом на финансовую грамотность и премиальный пользовательский опыт.
